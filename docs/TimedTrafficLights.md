# CSM TM:PE Sync - Timed Traffic Lights / Zeitgesteuerte Ampeln

This document is bilingual. German (DE) first, English (EN) follows.

---

## DE - Deutsch

Zeitgesteuerte Ampeln werden aktuell nicht unterstützt. Eine saubere Synchronisierung der Phasen würde im Multiplayer unverhältnismäßig viel Netzwerkverkehr verursachen, daher bleibt das Feature deaktiviert.

---

## EN - English

Timed traffic lights are currently unsupported. Keeping the phase sequence synchronized in multiplayer would generate disproportionate network traffic, so the feature remains disabled.
